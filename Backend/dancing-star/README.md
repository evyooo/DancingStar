```json
{
    "status": 200, // 민정 지정 status code
    "message": "민정 지정 response message",
    "response": {

    }

```

### Client-Exception

**[400]**

```json
{
    "status": 400,
    "message": "DancingID 잘못 입력 + [message]",
    "response": null
}
```




### Server-Exception

기본적으로 개발자 피드백 및 코드 해석이 필요함

**[501]** IOException

```json
{
    "status": 501,
    "message": "local IO 처리중 에러 + [message]",
    "response": null
}
```

local folder/file 읽기 및, 쓰기에서 발생하는 오류



**[502]** NcloudException

```json
{
    "status": 502,
    "message": "restAPI 연결 중 에러 + [message]",
    "response": null
}
```

ncloud poseEstimation or FaceRecognition 사용 중 발생하는 오류



**[504**] PythonException

```json
{
    "status": 504,
    "message": "python 파일 실행 중 에러 + [message]",
    "response": null
}
```

java에서 python 파일 실행 중 값 처리에 대한 에러
